# Project Summary

## Project Name

Banking Performance Analytics

## Objective

The objective of this project is to build an analytics dashboard that helps monitor banking performance across customer growth, CASA transactions, savings mobilization, credit repayment, and collateral risk.

## Dataset

The dataset is based on a banking SQL Server database that includes customer information, branch information, current account transactions, savings deposits, mobilization plans, loan contracts, loan repayments, and collateral records.

## Main Business Areas

1. Customer and branch analysis
2. CASA transaction analysis
3. Savings and mobilization plan analysis
4. Credit and repayment analysis
5. Collateral and risk exposure analysis

## Main Workflow

1. Build and inspect the SQL Server database.
2. Create relationships and ERD.
3. Create SQL views for Power BI.
4. Build DAX measures.
5. Design Power BI dashboard pages.
6. Analyze business insights.
7. Document the project for portfolio use.

## Key Skills Demonstrated

- SQL data preparation
- SQL aggregation and CTE
- Data modeling
- Fact and dimension view design
- DAX measure creation
- Power BI dashboard design
- Business insight writing
